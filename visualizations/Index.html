Summary Latitude Vs X

In this Project I used Google API and Open Weather API to get the weather details of 500+ cities randomly selected. With the help of the Citypy package I extracted 500+ random cities near the equator. Longitude and latitude were sent to openweather API to get the weather details.,

After getting the weather details, I created the summary of the weather per city and used a scatter plot to plot the results- City Latitude Vs Max Temperature, City Latitude Vs Humidity, City Latitude Vs Cloudiness and City Latitude Vs Wind Speed. After this I performed Linear regression on northern and southern hemisphere data.

After this, I performed Linear regression on northern and southern hemisphere's data for all the weather results received by the above API.
